# thePiratesCodingTest

## 환경 설정 및 실행 방법
>
> Java 버젼: 1.8 설치
> 
> Gradle기반의 SpringBoot 프레임워크
>
> 개발 DB : H2(In-Memory 모드 사용)
>
>
> **실행 순서**
>
> 1. 프로젝트 다운로드
> 
> 2. 인텔리제이 IDE에서 실행
> 
> 3. http://localhost:8080/swagger-ui.html 접속 후 API 테스트로 과제 확인
>
